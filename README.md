# OpenAML

OpenAML is an open-source AML analytics toolkit for transaction monitoring scenario testing, synthetic banking data, and alert explainability.

## Problem

AML teams, students, fintech companies, and compliance professionals often struggle to test transaction monitoring scenarios because realistic banking data is sensitive, expensive, or unavailable.

OpenAML aims to solve this by providing open-source tools to generate synthetic financial data, test AML scenarios, and explain why alerts are triggered.

## Who is this for?

- AML analysts
- Data analysts
- Data engineers
- FinTech teams
- Compliance consultants
- Students learning financial crime analytics

## Version 1 Scope

OpenAML Version 1 will focus on five transaction monitoring scenarios:

1. Structuring detection
2. High-velocity transaction detection
3. Round amount activity detection
4. Dormant account reactivation
5. High-risk geography activity
