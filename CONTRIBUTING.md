# Contributing to OpenAML

Thank you for your interest in contributing to OpenAML.

OpenAML aims to provide open-source tools for AML analytics, transaction monitoring scenario testing, and synthetic banking data generation.

## Ways to Contribute

- Report bugs
- Suggest features
- Improve documentation
- Submit code improvements
- Create example datasets
- Share AML use cases

## Development Principles

- Keep solutions simple and explainable
- Prioritize real-world AML problems
- Write clear documentation
- Add tests for new functionality

## Code of Conduct

Be respectful, constructive, and collaborative.

We welcome contributors from all backgrounds and experience levels.
