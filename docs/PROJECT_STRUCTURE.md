## Folder Purposes

### scenarios

Contains AML detection rules.

Examples:

- Structuring Detection
- High Velocity Detection
- Round Amount Detection

### models

Contains data models used by OpenAML.

Examples:

- Transaction
- Alert

### reporting

Creates alert reports and summaries.

### utils

Shared helper functions.
